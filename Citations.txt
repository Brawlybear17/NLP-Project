Ideas for checkStrength()

This method is about analyzing the user’s password.

Length check

If password is shorter than 8 characters → flag “Too short”

Number check

Loop through characters

If no digits → flag “Needs a number”

Uppercase / lowercase check

If all lowercase → flag “Add uppercase letter”

If all uppercase → maybe flag “Add lowercase letter”

Special character check

If no symbols like !@#$% → flag “Add a symbol”

Common password check

Convert input to lowercase

See if it exists in your common passwords ArrayList → flag “Too common”

Optional advanced idea

Count repeated characters (like "aaaaaa") → flag “Too repetitive”

Check sequences like "12345" → flag “Too predictable”
